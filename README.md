# Llama CoreML Extractor

CoreML conversion and inference toolkit for Llama 3.2 1B models.

## Usage

```bash
# Install dependencies
uv sync

# Convert model
uv run python llama_coreml_converter.py

# Run inference
uv run python llama_coreml_inference.py
```
